# 🌍 Student Globe-Trotter Pro 2026

**J'ai créé ce site pour aider les étudiants qui partent à l'étranger à comparer le coût de la vie dans 350 villes dans le monde — et surtout à savoir combien ça va leur coûter chaque mois.**

🔗 **[→ Voir le site](https://student-globe-trotter.vercel.app)**

---

## C'est quoi concrètement ?

Tu choisis une ville, tu entres ton budget mensuel, et l'app te calcule combien tu vas dépenser en logement, courses, restos, transports, soirées... Tu peux comparer 3 villes en même temps en mode Battle, les voir sur une carte monde, noter des infos personnelles sur chaque destination.

---

## Ce que tu peux faire

- 🏙️ Comparer **350 villes** sur 6 continents
- ⚔️ **Battle Mode** — 3 villes côte à côte
- 🗺️ **Carte interactive** cliquable
- 🛡️ **Indice de sécurité** (Numbeo 2024)
- 🗣️ **Niveau d'anglais** par ville (EF EPI 2023)
- 📝 **Notes perso** sur chaque ville
- ⭐ **Favoris**
- 🌐 Disponible en **français, anglais, espagnol**
- ☀️ Mode clair / sombre

---

## Stack

- **React 18** + **Recharts** pour les graphiques
- **SVG** pour la carte monde
- Zéro backend — tout tourne dans le navigateur

---

## Lancer en local

```bash
git clone https://github.com/Elio-code/Student-globe-trotter.git
cd Student-globe-trotter
npm install
npm start
```
