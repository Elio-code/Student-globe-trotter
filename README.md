# Student Globe-Trotter Pro 2026 🌍

Comparateur de coût de vie pour étudiants en mobilité internationale.

## ✨ Fonctionnalités
- **606 villes** sur 6 continents
- Carte interactive HD avec **vraies frontières de pays** (TopoJSON Natural Earth 50m)
- **Zoom & pan** à la molette ou au toucher
- Calculateur de budget personnalisé
- Mode sombre/clair, 3 langues (FR/EN/ES)
- Sécurité, niveau d'anglais, Battle Mode
- Favoris persistants

## 🚀 Installation

```bash
npm install
npm start
```

## 🌐 Déploiement (Vercel)

1. Push sur GitHub
2. Importer le repo sur vercel.com
3. Laisser les paramètres par défaut → Deploy

## 🏗️ Stack
- React 18 + Recharts
- TopoJSON (Natural Earth 50m) via CDN
- CSS-in-JS (zéro dépendances style)
